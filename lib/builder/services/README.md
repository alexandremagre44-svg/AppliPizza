# Builder B3 - Services

This directory contains the business logic and data services for the Builder B3 system.

## Purpose
- Handle Firestore operations for pages and blocks
- Manage draft/published states
- Support multi-resto configuration
- Provide data synchronization

## Future Contents
- `builder_firestore_service.dart` - Firestore CRUD operations
- `page_service.dart` - Page management
- `block_service.dart` - Block data operations
- `publish_service.dart` - Draft/publish workflow
- `restaurant_service.dart` - Multi-resto support

## Key Operations
- Save page drafts
- Publish pages
- Load pages for editing
- Sync across restaurants
- Version management
