# Builder B3 - Models

This directory contains the data models for the Builder B3 system.

## Purpose
- Define the structure of pages, blocks, and configurations
- Provide serialization/deserialization for Firestore
- Support multi-page, multi-resto architecture

## Future Contents
- `page_model.dart` - Page structure and metadata
- `block_model.dart` - Block definitions and configurations
- `block_types.dart` - Enum for different block types
- `builder_config_model.dart` - Builder configuration
- `restaurant_config_model.dart` - Multi-resto support
