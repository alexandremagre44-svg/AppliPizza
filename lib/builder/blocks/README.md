# Builder B3 - Blocks

This directory contains the block components for the Builder B3 system.

## Purpose
- Modular block system for page building
- Each block is a self-contained, reusable component
- Blocks can be added, removed, and reordered

## Future Block Types
- `text_block.dart` - Text content block
- `image_block.dart` - Image display block
- `button_block.dart` - Action button block
- `product_grid_block.dart` - Product listing block
- `hero_block.dart` - Hero banner block
- `category_list_block.dart` - Category navigation block
- `custom_html_block.dart` - Custom HTML content

## Block Structure
Each block should have:
- Widget for rendering
- Configuration model
- Editor interface
- Preview support
